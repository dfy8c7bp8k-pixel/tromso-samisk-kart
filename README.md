# Samiske stedsnavn – Tromsø

Interaktivt kart over samiske stedsnavn i Tromsø kommune, bygget med Leaflet og Kartverket,
med infotekst og lenker på både samisk og norsk.

## Innhold
- Samiske og norske stedsnavn
- Infotekst på samisk og norsk
- Lenker til offentlige Notion-sider
- Kartgrunnlag fra Kartverket

## Struktur
